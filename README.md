# NoSpaceTextView
解决TextView文本内容上下留白问题。NoSpaceTextView集成自TextView，用法和原生TextView一样，所有属性和方法均像往常一样使用。NoSpaceTextView 只是通过巧妙的办法，去除掉原生的留白问题。
